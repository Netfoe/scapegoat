# Project Instructions

## UI & UX Standards
- **Component Library:** DO NOT use any third-party component libraries (e.g., shadcn/ui, MUI). Use **native HTML components** exclusively.
- **Layout:** The frontend must use the **whole page width**. This is a professional productivity tool optimized for speed and information density.
- **Styling:** Use **modern, raw CSS** with well-defined, semantic classes defined in `index.css` or module stylesheets. **NEVER use Tailwind CSS.** **NEVER use inline styles (`style={{...}}`) on any element.** All styling must be driven by CSS classes.

## Tech Stack
- **Frontend:** React (TypeScript), Vite, modern pure CSS.
- **Backend:** Go (Golang).
- **Authentication:** ZITADEL.
